## Contribuer

Cette documentation est maintenue par la communauté. Pour modifier une page :
1. Trouvez le fichier `.mdx` correspondant dans `site/fr/docs/`.
2. Modifiez le contenu.
3. Proposez une Pull Request.

Envie d’aider ? Faites une PR sur GitHub (`/site/fr/docs/*.mdx`) ou ouvrez un ticket avec le nom de la page et votre suggestion. Les petites corrections (typos, liens cassés, étapes plus claires) sont appréciées.
