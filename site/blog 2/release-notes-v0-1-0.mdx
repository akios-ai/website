  <h1>AKIOS v0.1.0 Released</h1>
  <div class="post-author">
    <img src="https://github.com/aj14pm.png" alt="Author" class="avatar">
    <span>AJ</span>
  </div>
</header>

<div class="post-content">
  <p class="lead">
    We are thrilled to announce the first public preview of AKIOS v0.1.0. This release focuses on core stability and the fundamental sandboxing primitives.
  </p>

  <h2>Highlights</h2>
  <ul>
    <li><strong>Core Runtime:</strong> Stable release of the <code>akios-runtime</code> binary for macOS and Linux.</li>
    <li><strong>Policy Engine:</strong> Flexible YAML-based policy definitions for network and file access.</li>
    <li><strong>CLI Tools:</strong> New <code>akios</code> CLI for managing agent lifecycles.</li>
  </ul>

  <h2>Installation</h2>
  <p>
    You can install the latest version via Homebrew or by downloading the binary directly.
  </p>

  <pre><code>brew tap akios-os/tap
brew install akios

# Verify installation
akios --version</code></pre>

  <h2>Changelog</h2>
  <h3>Added</h3>
  <ul>
    <li>Initial support for WASM-based agents.</li>
    <li>Added <code>--strict</code> mode for paranoid security defaults.</li>
    <li>Documentation site launch (you're looking at it!).</li>
  </ul>

  <h3>Fixed</h3>
  <ul>
    <li>Fixed a race condition in the logger that caused dropped logs under high load.</li>
    <li>Corrected file permission handling for read-only mounts.</li>
  </ul>

  <h2>Contributors</h2>
  <p>
    A huge thank you to everyone who contributed to this release. We couldn't have done it without the community support.
  </p>
