### New in V1.1.0–V1.6.0

<div class="features-grid docs-cards">
  <a href="workflow-schema.html" class="docs-card">
    <h3 class="docs-card-title">Webhook & Database Agents →</h3>
    <p class="docs-card-text">Two new agents: webhook (Slack, Discord, Teams) and database (PostgreSQL, SQLite). 6 agents total. Added in v1.1.0.</p>
  </a>
  <a href="workflow-schema.html" class="docs-card">
    <h3 class="docs-card-title">Parallel Execution →</h3>
    <p class="docs-card-text">Run steps concurrently with parallel: blocks in workflow YAML. Plugin system for pip-installable agents. Added in v1.1.0.</p>
  </a>
  <a href="cli-reference.html" class="docs-card">
    <h3 class="docs-card-title">Dashboard & Reports →</h3>
    <p class="docs-card-text">akios dashboard for local compliance UI, akios run --report for HTML security posture reports. Added in v1.5.0–v1.6.0.</p>
  </a>
  <a href="cli-reference.html" class="docs-card">
    <h3 class="docs-card-title">Ollama Local Models →</h3>
    <p class="docs-card-text">Run Llama, Mistral, Gemma locally via Ollama — no API key needed. 7 LLM providers total. Added in v1.4.2.</p>
  </a>
  <a href="migration-guide.html" class="docs-card">
    <h3 class="docs-card-title">EnforceCore Integration →</h3>
    <p class="docs-card-text">Optional dependency: secret scanning, content rules, EU AI Act reports, Merkle bridge. pip install akios[enforcecore]. Added in v1.2.0.</p>
  </a>
  <a href="cli-reference.html" class="docs-card">
    <h3 class="docs-card-title">Audit Migrate & Prune →</h3>
    <p class="docs-card-text">Migrate JSONL to SQLite/PostgreSQL, enforce retention policies. Added in v1.5.0.</p>
  </a>
</div>

## Contributing
This documentation is community-maintained. To edit any page:
1. Find the `.mdx` file in `site/docs/`.
2. Edit the content.
3. Submit a PR!

Want to help? Open a PR on GitHub (`/site/docs/*.mdx`) or file an issue with the page name and suggested change. Small fixes (typos, broken links, clearer steps) are welcome.
