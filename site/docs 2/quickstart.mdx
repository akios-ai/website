## 1. Installation

AKIOS V1.0 provides a standalone binary for zero-setup execution.

### Download Binary (Recommended)

**Linux (x64)**
```bash
curl -L -o akios-linux-x64 https://github.com/akios-ai/akios/releases/download/v1.0.0/akios-linux-x64
chmod +x akios-linux-x64
```

**macOS (Universal)**
```bash
curl -L -o akios-macos-universal https://github.com/akios-ai/akios/releases/download/v1.0.0/akios-macos-universal
chmod +x akios-macos-universal
```

**Windows (x64)**
Download `akios-windows-x64.exe` from [GitHub Releases](https://github.com/akios-ai/akios/releases).

> **Other Methods**: See [Installation Guide](installation) for Pip (Python) or Docker instructions.

## 2. Create Your Project

Initialize a new project skeleton. This creates your configuration, templates, and directory structure.

```bash
# Linux
./akios-linux-x64 init my-project

# macOS
./akios-macos-universal init my-project

# Windows
.\akios-windows-x64.exe init my-project
```

Navigate into your new project:

```bash
cd my-project
```

---

## 3. Configure & Run

Run your first workflow. The **Interactive Setup Wizard** will automatically launch to help you configure your AI provider.

```bash
# Linux
../akios-linux-x64 run templates/hello-workflow.yml

# macOS
../akios-macos-universal run templates/hello-workflow.yml
```

**The Wizard will guide you to:**
1.  Select your AI Provider (OpenAI, Anthropic, Grok, Mistral, Gemini).
2.  Enter your API Key (securely stored in `.env`).
3.  Set budget limits (e.g., $1.00 max cost).
4.  Verify connectivity.

---

## 4. Explore the "Security Cage"

AKIOS is not just running the script; it's monitoring it.

### View Security Status
Check the status of your project and active security measures.

```bash
../akios-linux-x64 status --security
```

### Audit Logs
Every action is cryptographically logged. Check the audit trail:

```bash
cat audit/audit_events.jsonl
```

### Clean Up
Remove old run data when you're done.

```bash
../akios-linux-x64 clean --old-runs
```
