# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and inclusive environment for everyone contributing to the AKIOS project, regardless of experience level, background, or identity.

## Expected Behavior

- **Be respectful** – Treat all participants with consideration and professionalism
- **Be collaborative** – Work together constructively and share knowledge
- **Be clear** – Communicate clearly and provide helpful feedback
- **Be professional** – Focus on the technical merit of contributions

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Trolling, insulting comments, or unconstructive criticism
- Publishing private information without consent
- Any conduct that would be inappropriate in a professional setting

## Enforcement

Violations of this Code of Conduct may result in:
- Warning
- Temporary ban from project spaces
- Permanent ban from the project

## Reporting

If you experience or witness unacceptable behavior, please report it to:

**Email:** contact@akios.ai

All reports will be handled confidentially.

## Scope

This Code of Conduct applies to all project spaces, including:
- GitHub repositories and issue trackers
- Documentation and wikis
- Community forums and chat channels
- Project events and meetups
