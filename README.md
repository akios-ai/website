# AKIOS Website

This repository contains the source code for the public AKIOS website.

The website presents the AKIOS Open Runtime (GPL‑3) and AKIOS PRO, and is intended to be simple, fast, and easy to maintain.

## Project layout

- `site/` — HTML, CSS, MDX and static assets for the public site.
  - `site/index.html` — marketing home page.
  - `site/docs/` — documentation pages.
  - `site/blog/` — blog posts.
  - `site/community.html` — community page.
  - `site/legal.html` — legal page.
- `partials/` — shared HTML snippets injected into pages.
  - `header.html` — global navigation bar.
  - `footer.html` — global footer.

## Contributing

The website content is managed in the `site/` directory.

- **Documentation**: Edited in `site/docs/` (MDX).
- **Marketing Pages**: Edited in `site/` (HTML).

To contribute changes:

1. Edit the files in your local clone.
2. Submit a Pull Request.
